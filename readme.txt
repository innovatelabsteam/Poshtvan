== Description ==

Poshtvan is a powerful and easy-to-use support and ticketing plugin for WordPress. Designed to streamline customer support, this plugin enables site administrators to manage user queries efficiently through a structured ticketing system. With a user-friendly interface and robust features, Poshtvan ensures seamless communication between users and support teams.

== Key Features ==

* **Ticket Management:** Easily create, assign, and track support tickets.
* **User-Friendly Interface:** Intuitive design for both administrators and users.
* **Email Notifications:** Get notified about new tickets and responses.
* **Customizable Settings:** Adjust the plugin to fit your support workflow.
* **Role-Based Access:** Control who can submit and manage tickets.
* **Mobile Responsive:** Fully optimized for all screen sizes.

== Installation ==

1. Download and install the plugin from the WordPress Plugin Repository or upload the ZIP file manually.
2. Activate the plugin through the WordPress admin panel.
3. Navigate to the **Poshtvan Settings** page to configure the plugin.
4. Customize the settings as per your requirements.
5. Start managing customer tickets efficiently!

== Usage ==

* Users can submit tickets via the frontend support form.
* Administrators and support agents can manage tickets from the WordPress dashboard.
* Email notifications keep users and admins updated on ticket progress.
